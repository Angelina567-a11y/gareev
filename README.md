# Currency Converter (Конвертер валют)

Это простое GUI-приложение на Python, которое позволяет конвертировать валюты с помощью внешнего API, сохранять историю операций и работать с ней.

## Автор
Ваше имя и фамилия

## Требования
- Python 3.x
- Библиотеки: requests, tkinter (обычно входит в стандартную поставку Python)

## Как получить API-ключ
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Зарегистрируйтесь.
3. Войдите в свой аккаунт и создайте API-ключ в разделе "API Keys".
4. В коде замените значение `API_KEY` на ваш полученный ключ.

## Как запустить
1. Склонируйте репозиторий или скачайте файлы.
2. Убедитесь, что установлены необходимые библиотеки (`pip install requests`).
3. Запустите файл `main.py` (или другое имя файла с вашим кодом).

## Пример использования
- Выберите валюту "из" и "в".
- Введите сумму.
- Нажмите кнопку "Конвертировать".
- История операций отобразится ниже в таблице.

## Замечания
- История сохраняется в файл `history.json`.
- При повторном запуске история восстанавливается.
